# EncryptNotes

EncryptNotes - Зашифрованные заметки

## Что нового добавили в EncryptNotes

- 3 Функции

- Безопасность

- Регистрация / Вход

### Функции

- Создать заметки

- Посмотреть заметки

- Ввести Fernet-Ключ для расшифровки 

### Безопасность

- Fernet-Ключ для каждого пользователя

- Хэширование паролей

### Примечания

- Потеряв пароль вы не сможете войти в аккаунт! Вы потеряете доступ к заметками!

- Потеряв Fernet-Ключ вы не сможете расшифровать заметки! Потеряете естественный вид заметок
